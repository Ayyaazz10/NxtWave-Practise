# News Card Project

This project is a practice exercise to build a **news card layout** using **HTML, CSS, and Bootstrap**.

## What I Learned

* How to structure a webpage using semantic HTML.
* Linking an external CSS file and Google Fonts.
* Using **Bootstrap classes** for alignment and layout.
* Applying custom **CSS styles** for typography, spacing, and button design.
* Designing a neat card component with background and rounded corners.

## Tech Stack

* HTML
* CSS
* Bootstrap
