# Coding Cover Page

A small practice project to create a visually appealing coding cover page.

### What I Practiced
- HTML tags: `div`, `p`
- CSS properties: `background-image`, `border`, `padding`, `text-align`, `rgba()` for transparency
- Centering content using Bootstrap flex utilities
- Using Google Fonts (`Bree Serif`)
- Creating a simple, modern, and clean design
